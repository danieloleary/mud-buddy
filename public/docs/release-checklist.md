# Release Checklist

## Pre-release

- [ ] Backlog items for this release are closed or explicitly deferred.
- [ ] Version updated in `package.json` and `package-lock.json`.
- [ ] CHANGELOG.md updated.
- [ ] README quick links still match the live repo/site.
- [ ] Gallon-savings methodology reviewed.
- [ ] Public sharing checklist reviewed.
- [ ] Security/privacy docs reviewed if workflow changed.

## Test gate

- [ ] `npm ci`
- [ ] `npx playwright install chromium`
- [ ] `npm run validate`
- [ ] `npm run test:local-real-csv`

## Public package gate

- [ ] `public-site` generated by `npm run package:public`.
- [ ] Public ZIP contains only public-safe assets and approved synthetic SVGs.
- [ ] No private generated reports.
- [ ] No real CSV exports.
- [ ] No local paths.
- [ ] No deployment state files.
- [ ] `npm run test:redaction` passes.
- [ ] `npm run test:package-policy` passes.

## GitHub Pages gate

- [ ] Pages workflow passes.
- [ ] Live site opens at `https://danieloleary.github.io/mud-buddy/`.
- [ ] Landing page loads without console errors.
- [ ] Sample report link works.
- [ ] Approved SVG assets load, including `github-social-card.svg`.
- [ ] Subpath routing/assets work.
- [ ] Disclaimer is visible.
- [ ] Privacy language is visible.
- [ ] 1M-gallon mission language is present and framed as potential/helped-save.

## Release

- [ ] Tag created as `v1.0.0` after all gates pass.
- [ ] GitHub Release created from tag.
- [ ] Release notes copied from CHANGELOG.md.
- [ ] Known limitations listed.
