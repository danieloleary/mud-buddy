# Release Checklist

## Pre-release

- [ ] Backlog items for this release are closed or explicitly deferred.
- [ ] Version updated in `package.json` if this is a versioned release.
- [ ] `CHANGELOG.md` updated.
- [ ] README quick links match the live repo/site.
- [ ] Public sharing checklist reviewed.
- [ ] Security/privacy docs reviewed if workflow changed.

## Test Gate

```bash
npm ci
npx playwright install chromium
npm run validate
```

## Public Package Gate

- [ ] `public-site` generated by `npm run package:public`.
- [ ] `public-site` contains only public-safe assets.
- [ ] No private generated reports.
- [ ] No real CSV exports.
- [ ] No local paths.
- [ ] No deployment state files.
- [ ] `npm run test:redaction` passes.

## GitHub Pages Gate

- [ ] Pages workflow passes.
- [ ] Live site opens at `https://danieloleary.github.io/mud-buddy/`.
- [ ] Landing page loads without console errors.
- [ ] Sample report link works.
- [ ] Subpath routing/assets work.
- [ ] Disclaimer is visible.
- [ ] Privacy language is visible.

## Release

- [ ] Tag created as `vX.Y.Z` when appropriate.
- [ ] GitHub Release created from tag when appropriate.
- [ ] Release notes copied from `CHANGELOG.md`.
- [ ] Known limitations listed.
- [ ] X/LinkedIn/local community copy shared only after public URLs return `200`.
